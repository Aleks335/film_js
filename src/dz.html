Создать приложение (и фронэнд и бекенд) по поиску фильмов. В базе заранее подготовлена таблица в которой хранятся данные о фильмах. Например, каждый фильм это строка в базе с колонками : id, title (название), director_name(имя режиссера), year(год снятия). На фронтэнде мы по умолчанию выводим данные обо всех фильмах. Но присутствует возможность фильтрации. Например, показать фильмы снятые только после указанного года. Также существует возможность поиска по названию фильма. Также имеется возможность отсортировать фильмы по дате съемки. По сути можешь сделать фильтрацию по своему желанию
